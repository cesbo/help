La instalación de Astra es un proceso sencillo: solo copia un archivo binario a tu servidor.

## Requisitos

- Cualquier sistema operativo basado en Linux. Recomendamos usar Fedora 42.
- CPU x86 de 64 bits
- Conexión a internet en tu servidor para la validación periódica de la licencia en los servidores: `ls1.cesbo.com`, `ls2.cesbo.com`, y `ls3.cesbo.com`

:::note
Esta guía utiliza el comando **curl** para descargar archivos. La mayoría de las distribuciones de Linux tienen **curl** preinstalado.
:::

## Instalar Astra

<Steps>

1. **Abre la consola de tu servidor y descarga el archivo binario**

    ```sh
    curl -Lo /usr/bin/astra https://cesbo.com/astra-latest
    ```

2. **Configura el permiso de ejecución**

    ```sh
    chmod +x /usr/bin/astra
    ```

3. **Instalación completada**

    Puedes verificar la versión instalada de Astra ejecutando:

    ```sh
    astra -v
    ```

</Steps>

## Obtener Licencia

Se requiere una licencia válida para iniciar Astra en tu servidor. Puedes obtener una licencia de demostración gratuita o comprar una suscripción en nuestro sitio web: https://cesbo.com/astra-license

Para instalar la licencia, sigue las instrucciones en el correo electrónico con la nueva licencia.

## Iniciar Astra

Después de la instalación, registra un nuevo servicio en el systemd:

```sh
astra init
```

Este comando registra Astra como un servicio del sistema con el nombre `astra` y con el puerto de gestión `8000`. Para iniciar el servicio, ejecuta el comando:

```sh
systemctl start astra
```

Lee más sobre la gestión de servicios y el inicio de servicios adicionales en tu servidor: [Gestionar el Servicio Astra](/en/astra/admin-guide/manage-service).

## Habilitar autorun

Activa el inicio automático para tu servicio:

```sh
systemctl enable astra
```

Ahora el servicio `astra` se iniciará automáticamente al arrancar el sistema.

## Ajuste de Sistema

Para un mejor rendimiento de tu servidor Astra, ajusta la configuración de tu sistema. Puedes usar nuestro [script de ajuste](/en/articles/system/tune/) para configurar automáticamente estas configuraciones. Este script ajustará parámetros de red, modos de potencia del CPU, y otras configuraciones del sistema para mejorar el rendimiento de Astra.

## Iniciar sesión en la Interfaz Web

Después de instalar Astra, puedes acceder a su interfaz web usando un navegador web - Chrome, Safari o Firefox. El puerto predeterminado para la interfaz web es 8000. Así, la dirección de la interfaz será `http://tu-dirección-del-servidor:8000`. Para iniciar sesión en la interfaz web, utiliza el usuario y contraseña predeterminados, que ambos son "admin". Se recomienda cambiar la contraseña predeterminada para prevenir el acceso no autorizado.

Lee el siguiente artículo sobre [Interfaz Web de Astra](/en/astra/getting-started/web-interface)
