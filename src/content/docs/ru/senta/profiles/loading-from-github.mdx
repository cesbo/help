## Загрузка профиля с github

Вы можете загружать профили с github. Сначала перейдите на вкладку профиля и нажмите **Load newprofiles from github**.

![](https://cesbo.b-cdn.net/senta/help/streamer/Click_to_load.png)Затем нажмите **Show info**.

![](https://cesbo.b-cdn.net/senta/help/streamer/Click_to_show_info.png)Вы можете видеть информацию о профиле, и если хотите его добавить, нажмите **Accept**.

![](https://cesbo.b-cdn.net/senta/help/streamer/Click_to_accept.png)

## Загрузка профиля на github

<Steps>

1. **Сделать форк репозитория** : Перейдите по ссылке [https://github.com/cesbo/senta-streamer-profiles](https://github.com/cesbo/senta-streamer-profiles) и нажмите кнопку "Fork" в правом верхнем углу страницы. Это создаст копию репозитория в вашем аккаунте GitHub.

1. **Клонировать ваш форк** : Откройте ваш терминал (или Git Bash на Windows) и выполните команду `git clone https://github.com/your_username/senta-streamer-profiles`, заменив "your_username" на ваше имя пользователя в GitHub.

1. **Создать новую ветку** : Перейдите в клонированный репозиторий с помощью команды `cd senta-streamer-profiles`, затем создайте новую ветку, используя `git checkout -b add_profile`, где "add_profile" — это название вашей ветки.

1. **Добавить ваш файл** : Поместите ваш профиль в директорию /profiles в клонированном репозитории.

1. **Добавить файл в индекс и выполнить коммит** : Выполните следующие команды в вашем терминале:
    ```bash [terminal]
    git add profiles/your_file_name
    git commit -m "Added profile: %your_file_name%"
    ```
    Замените "your_file_name" на имя вашего файла.

1. **Отправить изменения в ваш репозиторий на GitHub** : Выполните команду `git push origin add_profile`, где "add_profile" — это название вашей ветки.

1. **Открыть Pull Request (PR)** : После отправки изменений перейдите на страницу вашего форкнутого репозитория на GitHub. Там вы должны увидеть кнопку "Compare & pull request" для вашей новой ветки. Нажмите на неё.

1. **Создать Pull Request** : Проверьте изменения, которые вы хотите внести, убедитесь, что всё правильно, и затем нажмите на кнопку "Create pull request".

1. **Ожидание обзора и слияния** : Другие участники проекта могут просмотреть ваш Pull Request, оставить комментарии или запросить изменения. После того, как ваш PR будет рассмотрен и одобрен, один из мейнтейнеров проекта объединит вашу ветку в основную ветку.

</Steps>

Вот и всё! Теперь ваш файл профиля должен быть добавлен в публичный репозиторий.
