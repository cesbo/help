---
title: Constant Bitrate (CBR)
---

Постоянный битрейт (CBR) — это метод заполнения пробелов в битрейте пустыми пакетами для достижения постоянной скорости, что требуется для DVB модуляторов.

Диаграмма ниже показывает поток без CBR, где битрейт изменяется и присутствуют пробелы:

![Диаграмма VBR](https://cdn.cesbo.com/help/astra/delivery/udp/vbr.png)

При включенном CBR поток заполняется пустыми пакетами для поддержания постоянного битрейта, что показано на диаграмме ниже:

![Диаграмма CBR](https://cdn.cesbo.com/help/astra/delivery/udp/cbr.png)

## Как настроить CBR

Чтобы настроить CBR в Astra, необходимо добавить параметр `cbr` к вашему UDP-адресу вывода. Параметр `cbr` доступен только для UDP-выводов. Например:

`udp://239.255.1.1#cbr=4000`

Это включит CBR с целевым битрейтом 4000 Кбит/с. Вы можете изменить значение битрейта в соответствии с вашими требованиями.

:::note
CBR не снижает битрейт. Если битрейт вашего потока превышает ограничение CBR, все лишние пакеты будут отброшены.
:::

## Как уменьшить битрейт

Опция cbr в Astra не уменьшает битрейт потока. Она только заполняет пробелы в битрейте пустыми пакетами для достижения постоянной скорости. Чтобы действительно уменьшить битрейт, требуется транскодировать поток. Это можно сделать с помощью таких инструментов, как [Senta](https://senta.tv) или FFmpeg.
