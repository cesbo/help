HTTP Play - это удобный и эффективный способ доступа ко всем вашим каналам одним нажатием. С помощью HTTP Play вы можете легко транслировать каналы по их ID, используя протоколы HTTP MPEG-TS или HLS. Эта функция упрощает процесс трансляции, предоставляя беспрепятственный доступ к вашим каналам и улучшая общий пользовательский опыт.

По умолчанию HTTP Play отключен. Чтобы активировать его, откройте: Настройки → HTTP Play.

![HTTP Play Options](https://cdn.cesbo.com/help/astra/delivery/http-hls/http-play/options.png)

- `Разрешить HTTP доступ ко всем потокам` - эта опция позволяет обеспечить доступ к каналам через протокол HTTP MPEG-TS
- `Разрешить HLS доступ ко всем потокам` - эта опция позволяет обеспечить доступ к каналам через протокол HLS

:::caution
Протокол HLS отключает запуск потоков по требованию и каналы будут работать постоянно. Также протокол HLS использует много оперативной памяти, около 4 Мб на 1 Мбит.
:::

- `Порт HTTP play` - параметр, позволяющий назначить порт, который будет использован для передачи плейлиста и медиа потоков. Он используется для разделения веб-интерфейса и плейлиста
- `Отключить TLS на порту HTTP Play` - эта настройка отключает использование HTTPS для этого медиаплейлиста
- `Путь до логотипов каналов` - параметр, используемый для указания пути к директории с файлами логотипов
- `Пользовательское имя для плейлиста` - параметр позволяет переименовать `playlist.m3u8` в произвольное имя
- `Организация плейлиста` - эта опция позволяет выбрать категорию для группировки каналов. Вы можете настроить группы в Настройки → Группы
- `playlist.m3u8` - это ссылка на плейлист каналов. Вы можете использовать плейлист на мобильных устройствах, компьютере, ТВ или приставке. Вы можете добавить данные аутентификации к плейлисту, например `https://your-server/playlist.m3u8?token=xxx`. [Подробнее](/en/astra/delivery-http/playlist/)

:::caution
По умолчанию аутентификация доступа отключена. Без аутентификации доступ к каналам не ограничен, что позволяет потенциально любому получить доступ к контенту. Чтобы ограничить доступ к вашим каналам, пожалуйста, рассмотрите возможность включения Авторизации Доступа или использования Сетевого Брандмауэра для ограничения доступа к вашему серверу. Это поможет обеспечить безопасность вашего контента и доступность только для авторизованных пользователей.
:::

## Логотип канала

Добавляя логотипы к вашим плейлистам каналов, вы предоставляете вашим зрителям более привлекательное и удобное визуально восприятие. Логотипы помогают пользователям быстро идентифицировать свои любимые каналы и делают навигацию по плейлисту более эффективной.

С помощью опции "Путь до логотипов каналов" вы можете указать расположение каталога изображений на вашем сервере. Astra загружает изображения с таким же именем, как и имя канала, и добавляет логотип к информации плейлиста.

1. Создайте директорию: `/var/lib/astra/tvg-logo`
2. В поле `Путь до логотипов каналов` укажите полный путь к этой директории: `/var/lib/astra/tvg-logo`
3. Сохраните логотипы каналов в директории. Используйте то же имя файла, что и имя канала. Например: `Travel Channel.png`
