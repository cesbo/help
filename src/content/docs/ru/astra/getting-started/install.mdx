Установка Astra — это простой процесс: просто скопируйте один бинарный файл на ваш сервер.

## Требования

- Любая операционная система на базе Linux. Мы рекомендуем использовать Fedora 42.
- x86 64-битный процессор
- Подключение к Интернету на вашем сервере для периодической проверки лицензии на серверах: `ls1.cesbo.com`, `ls2.cesbo.com` и `ls3.cesbo.com`

:::note
В этом руководстве используется команда **curl** для загрузки файлов. В большинстве дистрибутивов Linux **curl** предустановлен.
:::

## Установка Astra

<Steps>

1. **Откройте консоль вашего сервера и загрузите бинарный файл**

    ```sh
    curl -Lo /usr/bin/astra https://cesbo.com/astra-latest
    ```

2. **Установите разрешение на выполнение**

    ```sh
    chmod +x /usr/bin/astra
    ```

3. **Установка завершена**

    Вы можете проверить установленную версию Astra, запустив:

    ```sh
    astra -v
    ```

</Steps>

## Получение Лицензии

Для запуска Astra на вашем сервере требуется действующая лицензия. Вы можете получить демо-версию лицензии бесплатно или приобрести подписку на нашем сайте: https://cesbo.com/astra-license

Чтобы установить лицензию, следуйте инструкциям в письме с новой лицензией.

## Запуск Astra

После установки зарегистрируйте новую службу в systemd:

```sh
astra init
```

Эта команда регистрирует Astra как системную службу с именем `astra` и с портом управления `8000`. Чтобы запустить службу, выполните команду:

```sh
systemctl start astra
```

Узнайте больше о управлении службой и запуске дополнительных служб на вашем сервере: [Управление службой Astra](/en/astra/admin-guide/manage-service).

## Включение автозапуска

Включите автозапуск для вашей службы:

```sh
systemctl enable astra
```

Теперь служба `astra` будет автоматически запускаться при старте системы.

## Настройка Системы

Для повышения производительности вашего сервера Astra настройте параметры системы. Вы можете использовать наш [Скрипт настройки](/en/articles/system/tune/), чтобы автоматически сконфигурировать эти параметры. Этот скрипт будет настраивать сетевые параметры, режимы мощности процессора и другие конфигурации системы для улучшения производительности Astra.

## Вход в Веб-Интерфейс

После установки Astra вы можете получить доступ к ее веб-интерфейсу через веб-браузер - Chrome, Safari или Firefox. Порт по умолчанию для веб-интерфейса — 8000. Таким образом, адрес интерфейса будет `http://адрес-вашего-сервера:8000`.
Чтобы войти в веб-интерфейс, используйте стандартный логин и пароль, которые по умолчанию установлены как "admin". Рекомендуется сменить стандартный пароль, чтобы предотвратить несанкционированный доступ.

Прочтите следующую статью о [Веб-интерфейсе Astra](/en/astra/getting-started/web-interface).
